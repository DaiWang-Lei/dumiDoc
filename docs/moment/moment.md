# moment

## ISO 8601 格式

```js
//moment(string);
moment('2013-02-08'); //表示2013年2月8日
moment('2013-039'); // 表示的是2013年的第39天； 2013-02-08
moment('2013050'); //表示的是2013年的第50天； 2013-02-19
moment('2013W065'); //表示的是2013年的第六个星期的第五天；2013-02-08
moment('2013-02-08T09'); //表示2013年2月8日早上9点整
//moment(string)带格式
moment('12-25-1995', 'MM-DD-YYYY');
moment('12/25/1995', 'MM-DD-YYYY');
```
